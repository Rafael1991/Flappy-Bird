Flappy-Bird
===========

A flappy bird clone written in Java.

![demo](https://raw.githubusercontent.com/paulkr/Flappy-Bird/master/demo.png)


Build & Run
-----------

```shell
$ cd lib
$ javac FlappyBird.java && java FlappyBird
```
